# Web Tool Website

A Next.js application providing free online tools including:

- Privacy Policy Generator
- Image Compressor
- PDF to Word Converter

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
```

## Deploy

Deploy to Vercel, Railway, or Render for hosting.
